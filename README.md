# MindlinkRG

Aplicación web desarrollada en Java (JSP, Servlets, JDBC) que permite a los usuarios registrar, consultar, editar y eliminar sus emociones.

## Funcionalidades

- Registro de usuarios
- Inicio de sesión
- Registro de emociones
- Visualización de emociones
- Edición y eliminación de emociones

## Tecnologías utilizadas

- Java
- JSP / Servlets
- JDBC
- MySQL
- Apache Tomcat
- GitHub

## Base de datos

Nombre: mindlink_java

Tablas:
- usuario
- emocion

## Seguridad

Las credenciales de la base de datos se almacenan en un archivo `db.properties`, el cual está excluido del repositorio mediante `.gitignore`.
